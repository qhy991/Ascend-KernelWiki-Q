# Ascend C Kernel Launch Operator Project

Official CANN operator-development guide for the kernel launch project. It describes the project structure used to build and run vector, tiling-enabled vector, cube, and vector+cube fusion operator samples, including the CMake-based development flow, kernel launch configuration, debugging hooks, and profiling integration.
