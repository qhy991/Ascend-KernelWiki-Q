# Ascend C Operator Development API Overview

Official API overview for Ascend C class libraries. It groups the API surface into high-level compute APIs such as Matmul and Softmax, host-side tiling and prototype registration APIs, platform information APIs, and operator debugging helpers including DumpTensor, printf, assert, Trap, and CPU debug entry points.
