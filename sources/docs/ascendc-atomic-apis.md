# Ascend C Atomic Operation APIs

Official API overview listing SetAtomicMax, SetAtomicMin, SetStoreAtomicConfig, and GetStoreAtomicConfig. These APIs support reduction and split-output recipes that need controlled atomic writes to global memory.
