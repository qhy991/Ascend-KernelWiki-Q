# SGLang NPU Kernel — Ascend Backend

## Overview

SGLang's NPU kernel repository provides production-grade AscendC implementations of key inference kernels optimized for Ascend 910B accelerators. This is a primary reference for real-world AscendC kernel development in inference serving scenarios.

## Kernel Implementations

The repository includes optimized implementations for:

- **Attention kernels**: FlashAttention-style implementations with Cube/Vector overlap
- **Matrix multiplication (GEMM)**: High-performance matmul with NZ format handling
- **Softmax**: Numerically stable softmax with Vector unit optimization
- **LayerNorm**: Fused LayerNorm with bias and residual connection support

## Key Techniques Demonstrated

### Pipeline Scheduling
The codebase demonstrates effective use of independent Cube and Vector queues to maximize pipeline utilization. Attention kernels overlap matmul (Cube) with mask computation (Vector) for improved throughput.

### NZ Format Handling
Comprehensive examples of NZ format conversion, tiling strategies, and NZ-aware data layout management. Shows how to minimize format conversion overhead.

### Double Buffering
Memory transfer operations use double buffering to overlap GM↔UB transfers with compute, reducing MTE idle time.

## Learning Value

This repository is particularly valuable for understanding:

1. How production kernels handle variable sequence lengths in attention
2. Integration patterns with framework-level tensor abstractions
3. Performance-critical code organization in AscendC
4. Real-world pipeline scheduling for complex operators

Merged into main SGLang repository. Active development with performance optimizations for newer Ascend hardware generations.
