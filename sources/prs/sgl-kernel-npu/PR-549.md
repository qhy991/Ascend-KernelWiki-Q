# feat: add MXFP8 support for low_latency_dispatch

## Summary
This PR introduces MXFP8 (`at::kFloat8_e4m3fn`) and (`at::kFloat8_e8m0fnu`) support to DeepEP's `low_latency_dispatch` buffer logic using `moe_distribute_dispatch_v2` for Ascend environments.

## Architecture Changes
- Adjusts data type formats when `use_fp8` and `use_ue8m0` are enabled via `__DAV_C310__` architecture definitions.
- Expands the input and output datatypes of the `MoeDistributeDispatchV2` operator configuration to natively accept and return FP8 (E4M3 and E5M2/E8M0) tensor scales.
