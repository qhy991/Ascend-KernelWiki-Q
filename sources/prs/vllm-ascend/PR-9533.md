# vllm-ascend — AscendStore Hybrid/Mamba Aligned Prefix Cache

PR #9533 adds Hybrid/Mamba aligned prefix-cache support in AscendStore. It updates AscendStore connector/config/transfer/scheduler/worker paths and adds platform patches for Mamba config, scheduler, and Mamba utilities.

The core issue is that hybrid models have multiple KV/cache groups whose prefix-cache semantics are not identical. AscendStore must carry enough group metadata for aligned prefix-cache lookup and transfer rather than assuming one attention-only KV layout.
