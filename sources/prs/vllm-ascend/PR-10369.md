# vllm-ascend — Skip Pre-KV Graph Memory Profiling for DeepSeek-V4 Compressed Attention

## What Changed

PR #10369 skips pre-KV ACL graph memory profiling when `model_type == "deepseek_v4"` and compressed attention is enabled. The PR body says the temporary pre-KV capture path can hit `aclnnScatterNdUpdateV2` / 507015 before the normal KV-cache-backed graph path is available.

## Why It Matters

This is a lifecycle fix. It does not disable normal graph capture and replay. Instead, it recognizes that DeepSeek-V4 compressed attention cannot safely run the pre-KV memory-profiling capture before KV cache allocation.
