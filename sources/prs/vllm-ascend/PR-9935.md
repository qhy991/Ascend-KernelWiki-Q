# vllm-ascend — Keep DeepSeek-V4 DSA Forward Outside Piecewise Graph Capture

## What Changed

PR #9935 modifies `vllm_ascend/platform.py` so DeepSeek-V4's `vllm::dsa_forward` custom attention op is included in piecewise graph splitting behavior. The PR body explains that attention-like custom ops must be listed in `splitting_ops`; otherwise DSA attention may be captured inside a piecewise graph.

## Why It Matters

Graph capture is a runtime optimization boundary. A custom attention op can be correct in eager execution but unsafe or unsupported inside a captured graph segment. This PR records that DeepSeek-V4 DSA is such a boundary-sensitive op: it should remain outside the captured subgraph rather than being silently captured.
