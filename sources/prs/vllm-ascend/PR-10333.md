# vllm-ascend — Remove Manual DeepSeek-V4 Patch Environment Gate

## What Changed

PR #10333 removes the obsolete `VLLM_ASCEND_APPLY_DSV4_PATCH` environment variable from code, docs, tests, and coverage metadata. DeepSeek-V4 now uses the Ascend hybrid KV cache coordinator automatically when KV cache specs carry `model_version="deepseek_v4"`.

## Why It Matters

This PR is a productization step. It moves DeepSeek-V4 behavior from an opt-in environment flag to model-version-driven runtime routing, reducing configuration burden and preventing users from forgetting a required patch flag.
