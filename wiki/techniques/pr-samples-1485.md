# PR Insight: Ascend/samples #1485

**Title:** 【四川大学】【创新应用-性能预测器】项目训练代码及应用场景demo

## Overview
This PR adds a performance predictor project contributed by Sichuan University, including training code and application scenario demos. Performance prediction estimates model inference performance before deployment.

## Technical Significance
Performance prediction is valuable for optimizing model deployment decisions, resource allocation, and SLA planning. This sample demonstrates how to build and deploy performance prediction models on Ascend NPUs, potentially using historical training/inference data to predict throughput or latency.
