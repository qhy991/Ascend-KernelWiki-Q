**Title:** 添加resnet50_mindspore_picture样例

## Overview
This PR adds a ResNet-50 image classification sample using the MindSpore framework for processing picture input. The sample demonstrates how to run the classic ResNet-50 CNN architecture on Ascend hardware using MindSpore.

## Technical Significance
Adding ResNet-50 MindSpore samples provides developers with reference implementations for running widely-used CNN architectures on Ascend. ResNet-50 is a standard benchmark and production model, making this sample valuable for performance validation and learning.
