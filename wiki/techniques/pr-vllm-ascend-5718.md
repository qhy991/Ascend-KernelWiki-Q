# PR Insight: vllm-project/vllm-ascend #5718

**Title:** [Quantization] Support compressed tensors moe w8a8 int8 dynamic weight

## Overview
This PR adds support for compressed tensors quantization format to the VLLM Ascend engine, specifically enabling W8A8 Int8 dynamic weight quantization for MoE models. The implementation extends the `compressed_tensors.py` quantization module to handle compressed tensor formats generated by the LLM Compressor tool, and specifies W4A16 quantization configuration. The changes include new quantization configurations, updated documentation, and E2E test cases for MoE models.

## Technical Significance
This PR enables VLLM Ascend to work with compressed tensor quantization formats from the VLLM community toolchain, expanding support for MoE models with W8A8 Int8 dynamic weight quantization. The implementation adapts the Ascend engine to handle compressed tensor data structures and their specific weight storage format. This integration allows users to leverage LLM Compressor's quantization capabilities while maintaining compatibility with VLLM's Ascend backend for MoE inference.
