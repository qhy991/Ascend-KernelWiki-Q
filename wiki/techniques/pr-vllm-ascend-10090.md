# PR Insight: vllm-project/vllm-ascend #10090

**Title:** [Doc] move example from installation to quick_start

## Overview
This PR is a documentation update that reorganizes the vLLM-Ascend usage guides. Specifically, it relocates an example from the "installation" document to the "quick_start" document. 

## Rationale
By moving the example code into the Quick Start guide, the documentation becomes more logically structured. Users looking to understand how to run the model can find the necessary examples directly in the Quick Start section immediately after they finish the installation process. This improves the overall developer experience and readability.

## Key Changes
- **Documentation Migration:** An example usage snippet was moved from the installation instructions to the `quick_start` documentation.
- No functional code changes were made to the core system.
