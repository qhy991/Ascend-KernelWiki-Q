# PR Insight: Ascend/samples #1252

**Title:** 【加研贡献】DIY_小车_更新+无人机项目

## Overview
This PR updates DIY car and drone project samples contributed by the Huawei Ascend Research Institute. These are edge robotics applications running on Ascend NPUs.

## Technical Significance
Edge robotics (cars, drones) represent demanding real-time inference scenarios with strict latency, power, and compute constraints. These samples demonstrate how to deploy computer vision models for autonomous navigation, obstacle avoidance, or object tracking on Ascend edge devices like Atlas 200DK.
