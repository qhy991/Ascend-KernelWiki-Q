# PR Insight: vllm-project/vllm-ascend #38

**Title:** [Feature] Adapt to new quantization_config

## Overview
This PR adapts the quantization configuration to work with the new format generated by msmodelslim, removes unnecessary imports, and disables warnings when loading BasevLLMParameter. Changes affect quant_config.py and test utilities.

## Technical Significance
Compatibility update for the quantization pipeline. msmodelslim is Huawei's model compression toolkit, and this ensures vllm-ascend can load quantized models from the latest version. The test utility cleanup (30 deletions) simplifies the test infrastructure.
