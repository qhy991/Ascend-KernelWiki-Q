# PR Insight: vllm-project/vllm-ascend #743

**Title:** new modelslim quantization model  support v0.7.3

## Overview
This PR adds support for quantization models generated by new ModelSlim version by adding `quant_description` to `AscendQuantConfig`. The change enables vllm-ascend to work with updated quantization formats from ModelSlim, supporting the v0.7.3 version.

## Technical Significance
This update ensures compatibility with newer ModelSlim quantization tools, which is critical for running quantized models on Ascend hardware. The addition of `quant_description` provides necessary metadata for proper handling of quantized weights, enabling users to leverage quantization benefits without version conflicts.
