# PR Insight: Ascend/samples #1695

**Title:** 【外部平台适配案例新增】添加Opencv-MobileNet推理案例，Paddle-ResNet50推理案例

## Overview
This PR adds external platform adaptation cases, including an OpenCV-MobileNet inference example and a Paddle-ResNet50 inference example, demonstrating how to integrate Ascend inference with different frameworks.

## Technical Significance
External platform adaptation cases help developers understand how to use Ascend hardware from different programming frameworks. OpenCV integration enables computer vision workflows, while PaddlePaddle integration supports PyTorch model deployment through the Paddle ecosystem.
