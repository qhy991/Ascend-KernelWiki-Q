# PR Insight: vllm-project/vllm-ascend #3205

**Title:** [Feat]Qwen3 Moe supports npu_add_rms_norm_quant op by default, update op with norm bias

## Overview
This PR enables Qwen3 MoE to use the npu_add_rms_norm_quant operator by default instead of separate add_rms_norm and quantization operators. The operator also updates to handle non-zero norm bias generated by M4 asymmetric outlier suppression quantization method.

## Technical Significance
Operator fusion reduces computational overhead by combining RMS normalization and quantization into a single operation. Supporting norm bias is essential for M4 quantization, which produces non-zero bias values. This improves both performance and accuracy for quantized Qwen3 MoE models.
