# PR Insight: vllm-project/vllm-ascend #5889

**Title:** [Quantization][Feature] Support compressed tensors moe w4a8 dynamic weight

## Overview
This PR adds support for MoE models with W4A8 quantization using the LLM Compressor format. It adapts the vLLM Ascend engine to load and use compressed tensor quantization weights generated by the community's LLM Compressor tool.

## Technical Significance
Supporting the compressed_tensors format enables interoperability with the vLLM community's quantization ecosystem. MoE W4A8 dynamic weight quantization reduces memory footprint while maintaining accuracy. The PR implements weight loading logic in the compressed_tensors_config.py and w4a8.py modules, adds an example quantization script, updates the model list, and includes e2e tests for validation. This allows users to quantize MoE models with community tools and run them on Ascend NPU.
